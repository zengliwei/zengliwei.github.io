[ETL] SSIS 概述

什么是 SSIS

SQL Server Integration Services 是用于生成企业级数据集成和数据转换解决方案的平台，使用集成服务解决复杂的业务问题。Integration Services 可以提取和转换来自多种源（如 XML 数据文件、平面文件和关系数据源）的数据，然后将这些数据加载到一个或多个目标。Integration Services 包括：

  • 一套丰富的内置任务和转换；
  • 用于生成包的图形工具；
  • 用于存储、运行和管理包的 SSIS 目录数据库。

参考官网文档：https://learn.microsoft.com/zh-cn/sql/integration-services/sql-server-integration-services

安装 SSIS

软件需求：

软件版本 操作系统 ODBC 版本
SQL Server 2022 Windows Server 2016 或更高版本 Microsoft ODBC Driver 17 for SQL Server
SQL Server 2019 Windows Server 2016 或更高版本 Microsoft ODBC Driver 17 for SQL Server
SQL Server 2016 / 2017 Windows Server 2012 或更高版本 SQL Server Native Client

全新安装

  • 下载 SQL Server 安装程序：https://www.microsoft.com/zh-cn/sql-server/sql-server-downloads
  • 运行安装程序，选择“新建 SQL Server 数据库引擎实例”；
  • 在“功能选择”页面，选择“Integration Services”；
  • 继续安装，完成安装。

参考官网文档：https://learn.microsoft.com/zh-cn/sql/sql-server/install/hardware-and-software-requirements-for-installing-sql-server
